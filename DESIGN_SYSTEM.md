# 换换么 - 设计系统规范

## 📋 颜色系统

### 主色系
- **主绿色**: `#10b981` - 用于主要按钮、链接、强调元素
- **深绿色**: `#059669` - 用于按钮悬停、强调背景
- **浅绿色**: `#d1fae5` - 用于背景、标签

### 辅助色系
- **主蓝色**: `#0ea5e9` - 用于次要按钮、链接
- **深蓝色**: `#0284c7` - 用于蓝色悬停
- **浅蓝色**: `#cffafe` - 用于蓝色背景

### 中性色系
- **白色**: `#ffffff` - 卡片、输入框背景
- **浅灰**: `#f9fafb` - 页面背景
- **浅灰2**: `#f3f4f6` - 次级背景
- **中灰**: `#e5e7eb` - 边框、分割线
- **中灰2**: `#d1d5db` - 禁用状态
- **深灰**: `#9ca3af` - 辅助文字
- **深灰2**: `#6b7280` - 次要文字
- **深灰3**: `#4b5563` - 主要文字
- **黑色**: `#1f2937` - 标题、主要文字

### 功能色系
- **成功绿**: `#10b981` - 成功状态
- **警告橙**: `#f97316` - 警告状态
- **危险红**: `#ef4444` - 危险、删除操作

## 📝 字体系统

### 字体大小标准
- **超大标题**: `40rpx` - 页面主标题
- **大标题**: `32rpx` - 页面副标题
- **中标题**: `28rpx` - 卡片标题、章节标题
- **小标题**: `24rpx` - 小卡片标题
- **标题**: `20rpx` - 列表项标题
- **正文**: `18rpx` - 主要内容
- **正文小**: `16rpx` - 次要内容
- **标签**: `14rpx` - 标签、辅助信息
- **小文字**: `12rpx` - 时间、备注

### 字体粗细
- **加粗**: `700` - 标题
- **中粗**: `600` - 强调
- **正常**: `400` - 正文

### 行高
- **标题**: `1.2`
- **正文**: `1.6`
- **紧凑**: `1.4`

## 📐 间距系统

### 基础间距
- **超小**: `4rpx`
- **小**: `8rpx`
- **基础**: `12rpx`
- **中**: `16rpx`
- **大**: `20rpx`
- **超大**: `24rpx`
- **巨大**: `32rpx`

## 🔄 圆角系统

- **小**: `4rpx` - 小元素
- **基础**: `8rpx` - 按钮、输入框
- **中**: `12rpx` - 卡片
- **大**: `16rpx` - 大卡片
- **圆形**: `50%` - 头像、圆形按钮

## 🎨 阴影系统

- **浅阴影**: `0 1rpx 3rpx rgba(0, 0, 0, 0.08)`
- **基础阴影**: `0 2rpx 8rpx rgba(0, 0, 0, 0.1)`
- **中阴影**: `0 4rpx 12rpx rgba(0, 0, 0, 0.12)`
- **深阴影**: `0 8rpx 20rpx rgba(0, 0, 0, 0.15)`

## 🎯 使用指南

### 页面背景
- 主背景: `#f9fafb`
- 卡片背景: `white`

### 文字颜色
- 主要文字: `#1f2937`
- 次要文字: `#6b7280`
- 辅助文字: `#9ca3af`
- 反色文字: `white`

### 按钮
- 主按钮: 绿色背景 + 白色文字
- 次按钮: 蓝色背景 + 白色文字
- 文字按钮: 透明背景 + 绿色文字
- 禁用: 灰色背景 + 灰色文字

### 输入框
- 边框: `#e5e7eb`
- 背景: `white`
- 焦点: 绿色边框 + 绿色阴影
- 占位符: `#9ca3af`

### 标签/徽章
- 绿色标签: 绿色背景 + 深绿文字
- 蓝色标签: 蓝色背景 + 深蓝文字
- 橙色标签: 橙色背景 + 深橙文字

## 📱 响应式设计

### 断点
- 小屏: < 320rpx
- 中屏: 320-640rpx
- 大屏: > 640rpx

### 适配原则
- 移动优先
- 灵活布局
- 可读性优先

## ✨ 动画

### 过渡时间
- 快速: `0.1s`
- 标准: `0.2s`
- 缓慢: `0.3s`

### 缓动函数
- 默认: `ease`
- 进入: `ease-out`
- 退出: `ease-in`

## 🔍 检查清单

在提交任何样式改动前，请检查：
- [ ] 颜色值是否符合规范
- [ ] 字体大小是否符合规范
- [ ] 间距是否一致
- [ ] 圆角是否一致
- [ ] 阴影是否一致
- [ ] 文字对比度是否足够
- [ ] 移动设备上是否可读
- [ ] 功能是否完整
